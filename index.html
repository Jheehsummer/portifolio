O projeto de jokenpô utilizando HTML, CSS e JavaScript é um jogo onde o usuário pode jogar pedra, papel ou tesoura contra o computador. Na página, teremos os botões para o usuário escolher sua jogada (pedra, papel ou tesoura), uma área para exibir o resultado do jogo e um contador para mostrar a pontuação do usuário e do computador.
